# Clinicalab Platform - Design Style Guide

## Design Philosophy

### Visual Language
- **Medical Professionalism**: Clean, trustworthy, and authoritative design that instills confidence in healthcare decisions
- **Accessibility First**: Designed for all age groups with clear navigation and readable typography
- **Cultural Sensitivity**: Appropriate for Algerian market with Arabic/French language support
- **Modern Healthcare**: Contemporary design that feels advanced yet approachable

### Color Palette
- **Primary**: Medical Blue (#2563EB) - Trust, reliability, professionalism
- **Secondary**: Soft Teal (#0891B2) - Healthcare, cleanliness, calm
- **Accent**: Warm Orange (#F59E0B) - Urgency, call-to-action, energy
- **Neutral**: Cool Gray (#64748B) - Text, borders, subtle elements
- **Background**: Off-White (#FAFAFA) - Clean, medical environment feel
- **Success**: Medical Green (#059669) - Confirmations, positive actions
- **Warning**: Amber (#D97706) - Cautions, important notices

### Typography
- **Primary Font**: Inter (Arabic: Tajawal) - Modern, highly legible sans-serif
- **Secondary Font**: Cairo (Arabic) - Clean Arabic typography
- **Display Font**: Poppins (Arabic: Changa) - Bold headings and titles
- **Font Sizes**: 
  - Hero: 3.5rem (56px)
  - H1: 2.5rem (40px)
  - H2: 2rem (32px)
  - H3: 1.5rem (24px)
  - Body: 1rem (16px)
  - Small: 0.875rem (14px)

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth micro-interactions and element animations
- **ECharts.js**: Data visualization for lab comparisons and analytics
- **Splide.js**: Image carousels for lab galleries and testimonials
- **Typed.js**: Dynamic text effects for hero sections
- **Splitting.js**: Text animation effects for headings
- **p5.js**: Subtle background particle effects
- **Pixi.js**: Advanced visual effects for hero backgrounds

### Animation Effects
- **Hero Text**: Typewriter effect with gradient color cycling
- **Lab Cards**: Subtle hover lift with shadow expansion
- **Search Results**: Staggered fade-in animation
- **Booking Flow**: Progress indicator with smooth transitions
- **Comparison Tool**: Slide-in panels with elastic easing
- **Navigation**: Smooth underline animations for active states

### Header & Background Effects
- **Hero Background**: Subtle medical-themed particle system using p5.js
- **Section Backgrounds**: Clean gradients with soft color transitions
- **Card Shadows**: Medical-grade cleanliness with subtle drop shadows
- **Interactive Elements**: Soft glow effects on hover states

### Visual Hierarchy
- **Primary Actions**: High contrast buttons with medical blue
- **Secondary Actions**: Outlined buttons with hover fill transitions
- **Information Cards**: Clean white backgrounds with subtle borders
- **Warning/Alert States**: Color-coded with appropriate medical iconography

### Mobile Considerations
- **Touch Targets**: Minimum 44px for all interactive elements
- **Typography**: Scaled appropriately for mobile reading
- **Spacing**: Generous padding for comfortable mobile interaction
- **Navigation**: Collapsible menu with clear hierarchy

### Accessibility Features
- **Color Contrast**: Minimum 4.5:1 ratio for all text
- **Focus States**: Clear visual indicators for keyboard navigation
- **Screen Readers**: Proper ARIA labels and semantic HTML
- **Language Support**: RTL layout for Arabic content
- **Font Scaling**: Responsive typography that scales appropriately
