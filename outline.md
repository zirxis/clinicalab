## Page Breakdown

### 1. index.html - Main Landing Page
**Purpose**: Platform introduction and lab discovery
**Sections**:
- Navigation bar with logo and menu
- Hero section with medical imagery and value proposition
- Featured laboratories carousel
- Quick search functionality
- Service categories grid
- How it works section
- Testimonials slider
- Footer with platform information

**Key Features**:
- Advanced search with autocomplete
- Lab comparison preview
- Service category browsing
- Mobile-responsive design

### 2. search.html - Lab Search & Filter
**Purpose**: Comprehensive lab discovery and comparison
**Sections**:
- Search bar with filters
- Lab listing cards with ratings/prices
- Comparison tool sidebar
- Map view option
- Sort and filter controls

**Key Features**:
- Real-time filtering
- Side-by-side lab comparison
- Price and service comparison
- Location-based results

### 3. lab-details.html - Individual Lab Profile
**Purpose**: Detailed lab information and booking
**Sections**:
- Lab header with images and basic info
- Services and pricing table
- Equipment and credentials
- Doctor profiles
- Patient reviews
- Booking calendar

**Key Features**:
- Interactive service selection
- Real-time availability checking
- Direct booking integration
- Review and rating system

### 4. booking.html - Appointment Booking
**Purpose**: Complete booking process
**Sections**:
- Selected services summary
- Calendar date/time picker
- Patient information form
- Payment processing
- Confirmation details

**Key Features**:
- Multi-step booking flow
- Form validation
- Payment integration
- Confirmation system

### 5. profile.html - User Dashboard
**Purpose**: User account management
**Sections**:
- User information
- Booking history
- Upcoming appointments
- Health records
- Family members
- Settings and preferences

**Key Features**:
- Booking management
- Document upload
- Reminder settings
- Family account linking
